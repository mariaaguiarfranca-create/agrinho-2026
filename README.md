# semente = arvore= cidade

Projeto desenvolvido para o concurso agrinho 

🌱 Semente, Árvore e Cidade | Crescimento Sustentável

Uma landing page que demonstra a conexão entre natureza, urbanização e tecnologia. O projeto apresenta como pequenas ações (sementes) podem gerar grandes transformações ambientais, resultando em cidades mais verdes e sustentáveis.

# 🎯 Objetivo do Projeto

Demonstrar a importância do plantio de árvores para o desenvolvimento sustentável das cidades, destacando como a conscientização ambiental e a inovação tecnológica podem contribuir para a qualidade de vida da população.

# 🛠 Recursos Técnicos
# 🌱 Módulo Semente
Informações sobre germinação e plantio.
Dicas de preservação ambiental.
Animação mostrando o crescimento da semente.
# 🌳 Módulo Árvore
Benefícios das árvores para o meio ambiente.
Captura de carbono e produção de oxigênio.
Monitoramento do crescimento por meio de gráficos.
# 🏙️ Módulo Cidade
Planejamento urbano sustentável.
Áreas verdes e qualidade de vida.
Soluções inteligentes para cidades ecológicas.
# 🌎 Impacto Social e Ambiental

O projeto busca conscientizar a população sobre a importância das árvores no ambiente urbano, incentivando práticas sustentáveis e promovendo o equilíbrio entre desenvolvimento econômico e preservação ambiental.

# 🚀 Tecnologias Utilizadas
HTML5
CSS3
JavaScript
Responsividade para dispositivos móveis
Recursos de acessibilidade (alto contraste, aumento de fonte e leitor de texto)
